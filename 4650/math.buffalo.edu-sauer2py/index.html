Python Translations of the Sauer Numerical Analysis 2e Projects

These codes are written to facilitate the use of Timothy Sauer's textbook "Numerical Analysis" (2nd ed.)
with free open-source software.
To run them, you need Python (2.X, though modification for 3.X should be trivial) and the packages NumPy, SciPy and Matplotlib.
If you find errors or have suggestions, please contact ringland at buffalo.edu.
Codes for the 1st edition of the book are also available.
